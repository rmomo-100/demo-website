Quick goals

  • Anchor: Python + SQL
  • Cloud: Azure / Docker
  • Build: 3 micro-tools & 1 deployed MVP

Strategic plan

Focus on one anchor skill, iterate quickly, and validate with shipped MVPs. This plan gives you weekly tasks and a disciplined rhythm to make measurable progress.

Days 0–30

Foundation & momentum — set up environment, learn fundamentals, and ship your first mini-project.

  • Week 1: Install tools (VS Code, Git, Node, Python) and create GitHub repo
  • Week 2: Python fundamentals — pandas, CSVs, cleaning
  • Week 3: SQL practice — joins, window functions, CTEs
  • Week 4: Build first ETL script, push README, self-review

Days 31–60

Application — build real features, APIs, a small UI, and gather feedback.

  • Week 5: Learn FastAPI & create a simple REST endpoint
  • Week 6: Build ingestion pipeline & store cleaned data
  • Week 7: Create a dashboard (Streamlit/React)
  • Week 8: Start a vibe-coded micro-app; collect feedback

Days 61–90

Productize & ship — add AI features, polish UX, deploy, and promote.

  • Week 9: Add AI (OpenAI prompts, embeddings)
  • Week 10: Polish UI/UX & onboarding
  • Week 11: Deploy (Vercel/Netlify/GitHub Pages)
  • Week 12: Launch: landing page, case study, promotion
